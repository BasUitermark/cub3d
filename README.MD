### :gear: Installation

Download and build project:
```c
➜  ~ git clone https://github.com/w1codam/cub3d.git
➜  ~ make submodule\
➜  ~ cd libs/MLX42
➜  ~ cmake -B build # build here refers to the output folder.
➜  ~ cmake --build build -j4 # or do make -C build -j4

```
